# Markdown句法

## 什么是Markdown
它是一个文本编辑**“套路”**，与传统的Word编辑不同，它不需要右键菜单，也不需要图标按钮，纯靠手打。相信大家在热身作业里已经有了一些感受。

## 为什么要学习这个套路
因为：

![taolu](https://cloud.githubusercontent.com/assets/16900377/15090356/5134cf38-1458-11e6-8c69-e0fe2dc03cb9.png)

其实是因为：
* **快捷**：写文章的同时就能顺便编辑
* **页面简洁**：写完后生成的排版美观大方
* **方便做网页**
* **GitHub交流必备格式**
